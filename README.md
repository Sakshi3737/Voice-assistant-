📌 Project Overview
This project is a Voice Assistant built using Python that can perform various tasks through voice commands. It uses Speech Recognition for converting speech to text and Text-to-Speech (TTS) for responding back to the user.
The assistant can execute basic commands like opening websites, launching applications, and interacting with the system using voice.
🚀 Features:- 
        🎤 Speech-to-Text using Speech Recognition
        🔊 Text-to-Speech response
        🌐 Open websites like YouTube and Google
        💻 Launch applications (e.g., Chrome browser)
        🗣️ Interactive voice-based communication
        ⚡ Fast and simple command execution
🛠️ Technologies Used :- 
        Python
        SpeechRecognition
        pyttsx3 (Text-to-Speech)
        pyaudio
        webbrowser
        os module

⚙️ Installation
1. Clone the repository:
       Bash
       git clone https://github.com/Sakshi3737/voice-assistant.git

2. Navigate to the project folder:
       Bash
       cd voice-assistant

3.Install dependencies:
       Bash
       pip install -r requirements.txt
